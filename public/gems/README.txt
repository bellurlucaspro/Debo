DÉPOSER ICI LES 5 PIERRES TAILLÉES (visuels pince) DU HERO
===========================================================

Le carrousel du hero (GemShowcase) attend ces 5 fichiers, exactement nommés :

  gem-1.png   → Tourmaline pastèque (ronde rose/vert)
  gem-2.png   → Émeraude (ronde verte)
  gem-3.png   → Tourmaline bicolore (taille émeraude vert/rose)
  gem-4.png   → Tanzanite (poire bleue)
  gem-5.png   → Quartz mystique (octogone irisé)

Conseils :
- Fond TRANSPARENT (PNG) idéalement, ou fond uniforme identique sur les 5
  (pour un rendu cohérent quand les cartes défilent).
- L'image est alignée à droite : garder la pince qui « sort » par la droite,
  elle viendra alors affleurer le bord droit du conteneur.
- ~1024px de large suffit.

Les libellés/ordre se changent dans : src/components/storefront/GemShowcase.tsx
